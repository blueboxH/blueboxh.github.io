# 静态博客源码

> 此[静态博客](https://blueboxh.github.io)使用`hexo`搭建, 自动部署到`github`的`User`主分支, 此分支`source` 为源码分支, 博客可以在这个分支编写, 然后通过命令部署到主分支, 再把源码`push`到`source`分支, 这样即使换了一台电脑还是可以很方便的编写部署博客. 这里有一个我部署博客的[踩坑记](https://blueboxh.github.io/).
